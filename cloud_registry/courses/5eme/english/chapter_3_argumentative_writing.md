# Chapter 3 — Writing an Argumentative Paragraph (5th Grade)

## Objective
Learn how to defend an idea using **arguments** and **examples**.

## Key Notion to Remember
An argumentative paragraph contains:
- A **Thesis** (your opinion).
- 2 **Arguments** (reasons).
- 1 **Example** for each argument.

## Summary
- Thesis + 2 Arguments + Examples.
- Use connectors like "First", "Secondly", "Therefore".
